## Getting Started

Solo se debe clonar y ejecutar cargo run sobre cada carpeta del proyecto.

### Prerequisites

Actualmente estoy usando la version:

```bash
cargo --version # cargo 1.74.1
rustc --version # rustc 1.74.1
```

### Installing

```bash
git clone <this repo>
cd <any folder>
cargo run
```
